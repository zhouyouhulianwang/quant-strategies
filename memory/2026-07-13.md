- `setup_binance()` — 配置 Binance 数据源和订阅标的
- `get_prices()` — 获取所有最新价格
- `print_status()` — 打印完整状态

**3. 启动脚本** (`scripts/start_live_backend.py`)
- CLI 参数: `--mode` (paper/testnet/backtest), `--exchange` (alpaca/binance/all), `--symbols`, `--duration`
- 配置验证: 启动前检查 API Key 是否配置
