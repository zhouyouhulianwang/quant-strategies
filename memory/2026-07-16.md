## P0 Alpaca 模拟/实盘缺陷修复

**时间:** 2026-07-16 07:22 UTC
**提交:** `afa9450`
**范围:** 修复 Alpaca 执行链路 P0 严重缺陷

### 修复内容

| # | 缺陷 | 修复 |
|---|------|------|
| 1 | 实盘调仓不检查市场开盘时间 | `run_strategy.py` `run_live_rebalance()` 先调用 `self.executor.market_is_open()`，收盘时跳过并告警 |
| 2 | 实盘信号使用延迟 EOD 价格 | `generate_signals()` 新增 `live_mode` 参数，明确日志提示信号基于历史 EOD 收盘价，执行阶段使用实时价格 |
| 3 | 紧急平仓不检查市场状态 | `intraday_monitor.py` `_emergency_liquidation()` 和 `_liquidate_symbol()` 新增市场状态检查；收盘时记录告警 |
| 4 | 日内回撤基准 `daily_high_nav` 从不重置 | `_monitor_loop()` 检测日期变化，每天开盘时重置 `daily_high_nav` |
| 5 | `V14AlpacaExecutor` 缺少关键方法 | 在 `alpaca_executor.py` 补齐 `market_is_open()` / `liquidate_all()` / `submit_order()` / `get_account()` / `get_positions()` / `cancel_all_orders()` / `get_orders()` / `start_rebalance_session()` 透传方法 |
