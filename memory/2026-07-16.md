### 可用命令
```bash
# 回测（QuantConnect 数据优先）
python run_strategy.py --backtest --real-data

# 全自动实盘（信号→风控→下单→确认）
python run_strategy.py --live --paper

# 风险平价 + 盘中监控
python run_strategy.py --live --paper --monitor --weight-method risk_parity

# 定时调度
python scheduler.py
# cron: 30 15 * * * cd /path && python -c "from scheduler import run_once; run_once()"
```
