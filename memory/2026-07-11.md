## VIX 恐慌反转策略 (labs 项目)
- **时间**: 2026-07-11 UTC
- **策略原理**: 基于 VIX 盘中冲高 + RSI 分组的统计规律交易
  - 超卖组 (RSI<30): 恐慌抄底，次日反弹概率 80.4%，5日累计 +2.09%
  - 超买组 (RSI>70): 止盈信号，次日下跌概率 83.3%，5日累计 -1.26%
  - 中性组 (30≤RSI≤70): 无明确趋势，不交易
- **关键发现**:
  - A类 (VIX收盘≥30): 胜率 84.6%，5日 +2.38% → 满仓
  - B类 (VIX收盘<30): 胜率 66.7%，5日 +1.18% → 半仓
  - 纳指波动永远大于道指，道指防御性最强
- **策略参数**:
  - VIX 阈值: 20 (对应 VXX 代理 VIX=30)
  - 止盈: 2% / 止损: -2% / 最长持有: 5日
- **回测结果 (2020-2026)**:
  - 总收益: 27.00%
  - CAGR: 9.687%
  - Sharpe: 0.839
  - 胜率: 70% (76 胜 / 32 负)
  - 最大回撤: 8.1%
  - 交易次数: 108

## 云端 API 兼容性教训
- **小写 API 更安全**: 新 QuantConnect 项目必须使用 snake_case (`add_equity`, `rsi`, `set_holdings`, `liquidate` 等)
- **PascalCase 警告**: `self.AddEquity`, `self.RSI`, `self.Time` 等会在云端编译时产生 `has no attribute` 警告，尽管某些旧项目仍可用
- **VXX 确认可用**: 作为 VIX 指数代理，VXX 数据在 QuantConnect 云端可用，simple test 成功交易
- **云端日志不可见**: `lean cloud backtest` 终端输出只显示性能表格，策略日志不可见，需通过网页查看
