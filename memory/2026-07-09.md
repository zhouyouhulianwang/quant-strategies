### 已测试的优化方案

#### 1. 修复 except: pass ✅
- 所有7处 bare `except: pass` 替换为带日志的错误处理
- 修复误导注释："现金账户" → "Margin 账户"
- 回测无变化，代码健壮性提升

#### 2. 最大回撤控制 ❌
- 30%回撤时减仓到50%
- 结果：收益腰斩 4,977% → 2,546%
- 回撤仅降低 2.6%（45.5% → 42.9%）
- **已移除**：收益暴跌49% vs 回撤降低2.6%，完全不值得

#### 3. 市场过热保护 ❌
- SPY 3个月涨30%时降低总仓位到60%
- 结果：从未触发，无效果
- 原因：2020-2026期间SPY 3个月涨幅从未超过30%
- **已移除**

#### 4. 2018-2020 回测 ✅（熊市验证）
- 时段：贸易战 + 疫情崩盘 + 反弹
- 结果：268.7% / 夏普1.057 / 回撤52.5%
- 结论：策略在熊市中仍盈利，但收益大幅降低，回撤增大
- 验证了策略在危机中的生存能力

#### 5. 调仓阈值优化 ❌
- 测试 0.05（更敏感）和 0.15（更迟钝）
- 结果：0.10已最优
  - 0.05: 4,610% / 1.451 / 51.3%（过度交易）
  - 0.10: 4,977% / 1.568 / 45.5% ✅
  - 0.15: 3,955% / 1.474 / 45.5%（反应太慢）
