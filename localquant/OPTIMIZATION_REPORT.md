## 三、核心发现

### 3.1 策略特性
- **防御型策略**: 核心价值在于下行保护，而非收益增强
- **熊市表现出色**: 2022年跑赢SPY 20个百分点，回撤仅-3.20%
- **牛市表现疲软**: 2023-2024年跑输SPY 49个百分点
- **风险控制优秀**: 各时间段最大回撤均控制在-6.5%以内

### 3.2 最优参数（已验证）
```python
{
    "symbols": 180只跨行业标的,  # 非纯科技股
    "rebalance_freq": 20,        # 月度再平衡
    "max_position_pct": 0.10,    # 10%最大持仓
    "max_stocks": 10,            # 最多10只
    "stop_loss_pct": 0.08,       # 8%止损
    "trailing_stop_pct": 0.10,   # 10%跟踪止损
    "use_trend_filter": True,    # 启用趋势过滤
    "sector_rotation_enabled": True  # 启用行业轮动
}
```

### 3.3 关键教训
1. **简单 > 复杂**: SMA Cross策略夏普0.38，优于复杂的动量策略在部分时段
2. **频率至关重要**: 月度再平衡夏普0.63，远高于周度/日度
3. **过拟合风险**: 遗传算法参数在样本外失效，需用Walk-Forward验证
4. **无免费午餐**: 策略无法在牛熊两市同时跑赢基准
